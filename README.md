# 🌐 Nova CrossFlow 跨聊天指挥官

> AstrBot 插件 | 仅支持 aiocqhttp 平台 | by Nova (辉宝)

## ✨ 功能特性

**无需记忆任何命令**——直接用自然语言跟 AI 说话，AI 自动理解并执行跨聊天操作！

- **跨群发消息** — "你去XX群发一句XXX"
- **跨用户私聊** — "给QQ号123456789发个私聊说XXX"
- **智能临时会话** — 给非好友发私聊时，自动检测共同群组，通过临时会话通道发送
- **查询群列表** — "你在哪些群里？"
- **查找共同群** — "查一下能不能联系QQ号XXX"
- **安全白名单** — 管理员/指定用户才能使用，支持目标群号/用户白名单

## 🤖 使用方式

**不需要任何指令前缀！** 直接跟 AI 聊天就行：

```
你：去群 123456789 发一句"明天下午3点开会"
AI：[自动调用工具] 已成功向群 123456789 发送消息。

你：给 987654321 发个私聊说 记得交作业
AI：[自动调用工具] 已成功通过临时会话(via群111222333)向用户 987654321 发送消息。

你：你现在在哪些群里？
AI：[自动调用工具] 机器人加入了 5 个群：
    - XX群 (群号: 111, 50人)
    - YY群 (群号: 222, 30人)
    ...

你：查一下能不能联系 QQ号 555666777
AI：[自动调用工具] 该用户不是好友，但找到共同群：XX群 (群号: 111)。可以通过临时会话联系。
```

## 🔧 技术原理

插件通过 AstrBot 的 **LLM Tool Calling**（函数调用）机制工作：

1. 插件注册 4 个 LLM 工具：
   - `crossflow_send_group_message` — 向群发消息
   - `crossflow_send_private_message` — 向用户发私聊
   - `crossflow_get_group_list` — 获取群列表
   - `crossflow_find_common_group` — 查找共同群

2. 当用户用自然语言表达"给某某发消息"之类的意图时，LLM 自动识别并调用对应工具

3. 工具执行底层 OneBot 11 API 完成实际发送

## 🔒 权限控制

- 默认仅 **管理员** 可触发跨聊天工具
- 可在配置中添加 **白名单QQ号**，允许指定用户使用
- 可配置 **目标群号/用户白名单**，限制可发送的目标范围

## ⚙️ 配置项

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `allowed_sender_ids` | 允许使用的QQ号列表 | `[]`（仅管理员） |
| `allowed_target_group_ids` | 目标群号白名单 | `[]`（不限制） |
| `allowed_target_user_ids` | 目标用户白名单 | `[]`（不限制） |
| `enable_temp_session` | 启用临时会话 | `true` |
| `send_delay` | 发送延迟（秒） | `0.5` |
| `max_text_length` | 最大字符数 | `2000` |

## 🔄 临时会话工作原理

当要给 **非好友用户** 发私聊时：

1. 检查目标是否为好友 → 是则直接发送
2. 遍历机器人的所有群 → 检查目标用户是否在其中
3. 找到共同群 → 通过 `send_private_msg` + `group_id` 参数走临时会话通道
4. 没有共同群 → 返回失败原因

**可以在任何群或私聊中发起**，AI 自动寻找合适的通道！

## ⌨️ 手动命令

除了自然语言，也支持手动命令：

| 命令 | 说明 | 示例 |
|------|------|------|
| `/跨群 <群号> <消息>` | 向指定群发消息 | `/跨群 123456789 大家好` |
| `/跨私聊 <QQ号> <消息>` | 向指定用户发私聊 | `/跨私聊 987654321 你好` |
| `/跨流帮助` | 查看帮助信息 | `/跨流帮助` |

## ⚠️ 注意事项

- 仅支持 **aiocqhttp** 平台（NapCat/LLOneBot 等 OneBot 11 协议端）
- 需要 AI 模型支持 **Function Calling / Tool Use**
- 临时会话是否成功取决于 QQ 版本和协议端支持
- 建议配置白名单，避免被滥用
